# EzEat 🥗

EzEat es una plataforma diseñada para simplificar la gestión de pedidos y menús para restaurantes, enfocada en la eficiencia y la experiencia del usuario.

## Características
- 🚀 Interfaz moderna y rápida.
- 📱 Diseño totalmente responsivo.
- 🛠️ Fácil de personalizar.

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## Instalación
Para ver el proyecto localmente, simplemente clona el repositorio y abre `index.html` en tu navegador.

```bash
git clone https://github.com/[TU-USUARIO]/EzEat.git
```
